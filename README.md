# Cursor Landing Page Replica

This project is a front-end replica of the Cursor AI landing page.
It is built using HTML and CSS to practice layout design, grid systems, and responsive UI structure.

## Sections Recreated

The following sections from the original Cursor website were recreated:

- Top Navigation Bar (Logo, Nav Links, Buttons)
- Hero Section (Main headline and download button)
- Trusted By / Logos Section
- Feature Sections (3 main feature blocks with image + text)
- Testimonials Section
- Feature Cards Section
- Changelog Section
- Career Section
- Recent Highlights Section
- Call-to-Action Section
- Footer (Product, Resources, Company, Legal, Connect)
